# Сайт ремонтной мастерской (ПК / ноутбуки / периферия)

## Как открыть
- Откройте файл `index.html` двойным кликом (или перетащите в браузер).

## Страницы
- `index.html` — главная
- `services.html` — услуги
- `prices.html` — цены
- `about.html` — о мастерской (процесс, отзывы, FAQ)
- `contacts.html` — контакты и форма заявки

## Что поменять под себя
### Контакты (телефон/почта)
Файл: `assets/js/main.js`
- `CONTACTS.phone` и `CONTACTS.phonePretty`
- `CONTACTS.email`

### Адрес и график
Файл: `contacts.html`, блок **Контакты**.

### Цены и тексты
Файлы:
- `prices.html` — цены
- `services.html` — услуги
- `about.html` — процесс, отзывы и FAQ

## Структура
- `index.html` и `*.html` — разметка страниц
- `assets/css/styles.css` — стили и адаптив
- `assets/js/main.js` — меню, FAQ, год в футере, отправка заявки (почта/буфер)
